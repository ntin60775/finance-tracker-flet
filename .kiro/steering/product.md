# Product Overview

Finance Tracker Flet - десктопное приложение для управления личными финансами.

## Core Features

- **Транзакции**: учёт доходов и расходов с категоризацией
- **Кредиты и займы**: управление кредитами, займодателями, графиками платежей
- **Плановые транзакции**: периодические транзакции с правилами повторения (RecurrenceRule)
- **Отложенные платежи**: управление будущими платежами с приоритетами
- **План-факт анализ**: сравнение плановых и фактических транзакций
- **Прогноз баланса**: визуализация будущего баланса с учётом планов
- **Календарь**: визуальное представление транзакций и платежей

## Data Management

- **Экспорт/импорт**: базовый функционал в JSON (публичный)
- **Облачная синхронизация**: расширенный функционал через Git submodule (приватный)

## User Data Location

Все пользовательские данные хранятся в `.finance_tracker_data/`:
- `finance.db` - SQLite база данных
- `config.json` - настройки пользователя
- `logs/` - логи приложения
- `exports/` - экспортированные файлы

Директория создаётся:
- В режиме разработки: в корне проекта
- В режиме .exe: рядом с исполняемым файлом

## Distribution

- Режим разработки: `python -m finance_tracker`
- Standalone: PyInstaller создаёт `FinanceTracker.exe` (~50-70 MB)
- Портативность: все данные в `.finance_tracker_data/` рядом с .exe
