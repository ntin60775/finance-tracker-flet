Современная минималистичная иконка приложения для трекера личных финансов.
Концепция: "Цикл потока с Золотым щитом и Ростком на земле".
Визуальные элементы:
1. Внешнее кольцо: Круговой цикл потока, состоящий из двух изогнутых стрелок РАЗНЫХ цветов:
   - Стрелка дохода: Синий/Голубой (символизирует приток).
   - Стрелка расхода: Красный/Оранжевый (символизирует отток).
2. Центр: Блестящий Золотой Щит (#FFD700), символизирующий защиту капитала.
3. Внутреннее ядро: Внутри щита находится круг цвета Земли/Почвы (Коричневый).
4. Сердцевина: Ярко-зеленый росток растения, растущий из коричневого круга земли, символизирующий рост.
Стиль: Плоский дизайн (Flat) с легкими градиентами/эффектом блеска для щита, чистая геометрия.
Фон: Прозрачный.
Цвета: Золотой, Коричневый, Зеленый, Синий, Красный.
